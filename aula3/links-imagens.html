🔗 Links e Imagens

O HTML conecta o mundo: hiperlinks levam o usuário a outras páginas ou trechos, e imagens contam histórias sem palavras. Explore a anatomia das tags, monte links ao vivo e experimente no playground.

▸ anatomia clicável
Alterne entre <a> e <img>

O elemento âncora precisa de href (destino). O texto visível fica entre abertura e fechamento — é o que o usuário clica.

<a href="https://exemplo.org">
abertura + destino
→
Texto do link
conteúdo clicável
→
</a>
fechamento

<img> é uma tag vazia (não tem </img>). Use src para o arquivo e alt para descrever a imagem (acessibilidade e SEO).

<img src="foto.jpg" alt="Descrição">
um único bloco — tag completa

✏️ EXERCÍCIO 03

Página “Portfólio em uma tela”

No playground, monte um HTML que use pelo menos um link externo, um link interno (#), uma imagem com alt e um parágrafo.

  • 1
    Título — <h1> com seu nome ou apelido criativo.
  • 2
    Link externo — site que você recomenda (documentação, blog, etc.).
  • 3
    Imagem — use <img> com src (URL válida) e alt que descreva a imagem.
  • 4
    Âncora — um id em um subtítulo e um link href="#…" apontando para ele.
