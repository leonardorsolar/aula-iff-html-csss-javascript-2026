# aula-iff-html-csss-javascript-2026

Repositório com exemplos básicos de **HTML** para a disciplina de Programação para Web (IFF).

## Conteúdo

- Estrutura básica de página HTML
- Títulos e textos
- Links e imagens
- Formulários
- Exercícios de treino

## Arquivos principais

- `index.html`
- `estrutura-basica.html`
- `titulos-e-textos.html`
- `links-imagens.html`
- `formulario.html`
- `css-interativo-iff.html`
- `treino.html`

## Como visualizar

1. Abra a pasta do projeto no VS Code.
2. Clique em qualquer arquivo `.html`.
3. Abra no navegador (duplo clique no arquivo ou usando extensão Live Server).

## Objetivo

Praticar os fundamentos iniciais de desenvolvimento web com HTML (e, em etapas seguintes, CSS e JavaScript).
