▸ dicas de ouro

✅ Sempre faça

  • Indente o código com espaços ou tab
  • Feche todas as tags que abre
  • Teste no navegador com frequência
  • Comece simples e vá adicionando
  • Use nomes descritivos para classes e IDs

❌ Evite fazer

  • Copiar código sem entender o que faz
  • Usar letras maiúsculas nas tags HTML
  • Misturar muitos estilos inline
  • Desistir no primeiro erro — é normal!
  • Pular etapas para chegar ao final
