📋 Formulários HTML

Aprenda a criar formulários para coletar dados do usuário. Campos de texto, senhas, seleções, botões e muito mais — tudo com as tags certas.

▸ entenda o conceito
O que é um Formulário HTML?

Formulários são usados para coletar informações do usuário e enviá-las para um servidor. Login, cadastro, pesquisa, pagamento — tudo isso usa formulários!

🔑 A tag principal: Todo formulário começa com <form> e termina com </form>. Dentro dela ficam todos os campos (inputs).

▸ atributos importantes
Clique em um card para carregar no playground
placeholder=""
Dica dentro do campo antes de digitar.

Texto de exemplo que aparece quando o campo está vazio. Desaparece ao digitar.

required
Campo obrigatório — bloqueia envio.

Impede o envio se o campo estiver vazio. O navegador exibe mensagem de erro.

disabled
Campo desabilitado (cinza).

Desativa o campo. O usuário não consegue clicar nem digitar.

value=""
Valor padrão já preenchido.

Define o valor inicial do campo quando a página carrega.

name=""
Identifica o campo no envio.

Nome do campo que é enviado junto com o valor para o servidor. Essencial!

<label for="">
Rótulo associado ao campo.

Conecta o texto ao campo pelo atributo for. Clicando no label foca o input.

⚠️ Atenção: O atributo name é obrigatório para que o valor do campo seja enviado com o formulário. Campos sem name são ignorados no envio!

🔐 Login simples

<form> <h2>Login</h2> <label for="u">Usuário:</label> <input type="text" id="u" name="usuario" placeholder="seu usuário" required> <br><br> <label for="s">Senha:</label> <input type="password" id="s" name="senha" required> <br><br> <button type="submit">Entrar</button> </form>
Formulário de login com usuário e senha.

📝 Cadastro básico

Cadastro com nome, e-mail e data de nascimento.

✏️ EXERCÍCIO 03

Crie um formulário de cadastro de aluno!

Use o playground acima para montar um formulário completo. Use pelo menos 5 tipos de elementos diferentes desta aula.

  • 1
    Cabeçalho do formulário — Use <h2> com o título "Cadastro de Aluno".
  • 2
    Campo de nome — <label> + <input type="text"> com placeholder e required.
  • 3
    Campo de e-mail — Use type="email" para validação automática.
  • 4
    Seleção de turno — Use type="radio" com opções Manhã, Tarde e Noite (mesmo name nos três!).
  • 5
    Escolha de curso — Use <select> com pelo menos 3 <option>.
  • 6
    Checkbox de termos — type="checkbox" com label "Li e aceito os termos".
  • 7
    Botão de envio — <button type="submit"> com o texto "Realizar Cadastro". 
<form>
  <h2>Cadastro de Aluno</h2>

  <label for="nome">Nome completo:</label><br>
  <input type="text" id="nome" name="nome"
         placeholder="Digite seu nome completo" required>
  <br><br>

  <label for="email">E-mail:</label><br>
  <input type="email" id="email" name="email"
         placeholder="seu@email.com" required>

  <input type="radio" name="turno" value="M"> Manhã
  <input type="radio" name="turno" value="T"> Tarde
