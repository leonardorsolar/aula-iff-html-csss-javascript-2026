Etapa 1 — Primeiro exercício inline (HTML + CSS)

Conceito HTML + CSS

Primeiro, você vai criar o elemento no HTML e depois estilizar no CSS. Isso ajuda a entender a ordem de construção: 1) cria a tag e 2) aplica o estilo.

O que e o atributo style?

Tags HTML permitem atributos, que sao como "configuracoes" para cada elemento. O atributo style aplica CSS diretamente na tag, sem precisar de arquivo separado.

Ele e util para testes rapidos e exemplos curtos. Em projetos maiores, prefira CSS separado para manter o codigo organizado e facil de manter.

Exemplos de style

  • color: blue;
  • color: rgb(0, 0, 255);
  • color: #0000ff;
  • text-align: center;
  • text-align: justify;
  • text-align: left;
  • font-size: 40px;

Tarefa: no campo HTML, crie na tag h1 um atributo style. No CSS, deixe azul, tamanho 40px e centralizado.

Etapa 1.1 — Ajustando um <p> (Paragrafo)

Conceito HTML + CSS

Agora vamos repetir a mesma logica da etapa 1: primeiro criar o elemento em HTML e depois ajustar o estilo no CSS.

HTML base

<p>Este e um paragrafo de teste.</p>

Exercicio: aplique no paragrafo:

  • cor cinza
  • tamanho 18px
  • fonte Arial
  • alinhamento justificado

Exemplos de style

  • color: gray;
  • font-size: 18px;
  • font-family: Arial;
  • text-align: justify;

Modo: inline direto no atributo style da tag p.

Digite o HTML acima e clique em Testar.
✅ Excelente! Voce ajustou o paragrafo corretamente.

Etapa 1.2 — Estilizando um <input>

Conceito HTML + CSS

Vamos seguir o mesmo modelo da etapa 1.1: voce escreve o HTML e aplica o estilo inline no proprio elemento.

Exercicio: aplique no input:

  • altura 40px
  • borda preta
  • cantos arredondados

Exemplos de style

  • width: 300px;
  • height: 40px;
  • border: 1px solid black;
  • border-radius: 8px;

Modo: inline sem id, direto no atributo style da tag input.
