▸ as 5 tags essenciais
<!DOCTYPE html>

Avisa ao navegador que este é um documento HTML5. Sempre na primeira linha.

<html>

Elemento raiz. Envolve todo o conteúdo da página. Tudo fica dentro dele.

<head>

Informações sobre a página — invisíveis para o usuário: título, estilos, meta.

<title>

O texto que aparece na aba do navegador (não na página em si).

<body>

Tudo que o usuário vê na tela fica aqui: títulos, textos, imagens, botões.

🏠 Estrutura mínima

<!DOCTYPE html> <html> <head> <title>Minha Página</title> </head> <body> <h1>Bem-vindo!</h1> <p>Minha primeira página.</p> </body> </html>
A estrutura mais simples possível. Todo site começa assim.

🎨 Com estilo no head

<!DOCTYPE html> <html> <head> <title>Com Estilo</title> <style> body { background: #1e293b; color: white; } h1 { color: #06b6d4; } </style> </head> <body> <h1>Página com estilo!</h1> <p>O CSS fica no head.</p> </body> </html>
O CSS pode ficar dentro de uma tag style no head.

📋 Página completa

<!DOCTYPE html> <html lang="pt-BR"> <head> <meta charset="UTF-8"> <title>Página Completa</title> </head> <body> <h1>Título Principal</h1> <p>Primeiro parágrafo.</p> <p>Segundo parágrafo.</p> </body> </html>
Versão profissional com lang e charset declarados.

🧠 Explicação linha a linha (para dev júnior)

  • <!DOCTYPE html> — Diz ao navegador que o documento usa HTML5.
  • <html lang="pt-BR"> — Inicia a página e define o idioma principal para português do Brasil (ajuda SEO e acessibilidade).
  • <head> — Área de configurações da página (não aparece no conteúdo visível).
  • <meta charset="UTF-8"> — Permite acentos e caracteres especiais corretamente (ç, ã, é).
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> — Faz a página se adaptar bem em celular e tablet.
  • <title>Documento</title> — Texto que aparece na aba do navegador.
  • </head> — Fecha a área de configurações.
  • <body> — Início do conteúdo visível: títulos, textos, imagens, botões etc.
  • </body> — Fecha o conteúdo visível da página.
  • </html> — Fecha o documento HTML.
✏️ EXERCÍCIO 01

Hora de praticar!

Use o playground acima para completar as tarefas abaixo. Cada uma vai reforçar um conceito diferente da estrutura básica do HTML.

  • 1
    Altere o título da aba — Mude o texto dentro de <title> para o seu nome. Observe a mudança na aba (no preview, veja o título no topo).
  • 2
    Adicione mais conteúdo no body — Escreva mais um parágrafo <p> com seu curso ou profissão desejada.
  • 3
    Adicione uma tag style — Dentro do <head>, adicione <style> body { background: lightblue; } </style> e veja a página mudar de cor.
  • 4
    Atributo lang — Adicione lang="pt-BR" dentro da tag <html>. Isso é boa prática de acessibilidade! 
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>João Silva</title>
  <style>
    body { background: lightblue; }
  </style>
</head>
<body>
  <h1>Bem-vindo!</h1>
  <p>Esta é minha primeira página web.</p>
  <p>Sou estudante de Desenvolvimento Web.</p>
</body>
</html>
